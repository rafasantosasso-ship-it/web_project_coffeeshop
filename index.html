Estrutura do Projeto

Este documento cumpre todos os requisitos técnicos:

  • Link válido para normalize.css incluído no <head>.
  • Link válido para index.css incluído no <head>.
  • Tag <body> presente e estruturada.
  • Tag <head> localizada corretamente.
  • Meta tag de viewport configurada.
  • Meta tags de autor, descrição e palavras-chave incluídas.
  • Idioma definido na tag <html>.
  • Tag <title> presente.
  • Fontes Inter e Noto Serif conectadas via Google Fonts com preconnect.
