# Triple Espresso

Coffee Shop Website

Este projeto é um site de cafeteria desenvolvido utilizando HTML5 e CSS3, com foco na construção de layouts modernos e organização de código seguindo a metodologia BEM
O objetivo do projeto é praticar estruturação de páginas, organização de estilos e boas práticas de desenvolvimento front-end.

Tecnologias Utilizadas

HTML5
CSS3
Metodologia BEM

Funcionalidades

Layout estruturado com múltiplas seções
Navegação simples e clara
Seção de receitas
Formulário de reserva
Estrutura organizada com BEM

Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:
Organização de CSS com metodologia BEM
Uso de Flexbox para layout
Estruturação de páginas HTML semânticas
Separação de estilos por componentes
Uso de arquivos CSS modulares
